# Acknowledgements

MendCode is built on the opencode codebase and keeps the opencode MIT license notice intact.

This repository is not intended to imply that MendCode is a simple unmodified fork. MendCode is a downstream project and derivative work with substantial product/runtime additions, including:

- the `mend` public CLI surface
- MendCode-owned configuration and control-plane routing
- setup/onboarding flows
- reusable runtime packages and registry flows
- mflow coordination and edit-lock integration
- optional TSM/worktree orchestration
- model-role projection
- terminal UI profile, prompt chrome, widgets, and ASCII/mascot customization
- MendCode docs, installer, and release scripts

The opencode project deserves credit for the original runtime foundation. MendCode's goal is to build a distinct terminal-first coding product on top of that foundation while preserving attribution and license obligations.
