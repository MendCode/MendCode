# Support

Use the right channel so the answer helps future users.

## Questions

Use GitHub Discussions for:

- setup help
- package sharing
- TUI customization
- plugin/widget examples
- mflow and worktree usage
- model/provider configuration

## Bugs

Use GitHub Issues for reproducible bugs. Include:

- MendCode version
- OS and terminal
- install method
- expected behavior
- actual behavior
- logs, screenshots, or reproduction steps

## Security

Use [SECURITY.md](SECURITY.md) for vulnerabilities or suspected secret leaks.

Do not post private security details publicly.
