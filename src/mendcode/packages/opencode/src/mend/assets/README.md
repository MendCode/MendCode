# MendCode Runtime Assets

Runtime-owned assets live beside the MendCode source that consumes them.

- `tui/default-profile.json` and `tui/schema.json` are live product defaults for the MendCode TUI profile pipeline.
- `prompt-sources/**` are licensed OSS prompt evidence snapshots used by focus/prompt policy tooling. They are evidence inputs, not upstream identity claims.

Legacy render proof fixtures were archived to `docs/evidence/tui/` and are not runtime dependencies.
